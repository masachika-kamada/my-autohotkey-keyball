# My AutoHotkey Keyball

## 概要

* Windows のキーボードをカスタマイズできる AutoHotkey を使用してエンジニアとして必要な操作を効率的に実現
* 過去作 ([my-autohotkey-optimized](https://github.com/masachika-kamada/my-autohotkey-optimized)) をもとに [Keyball44](https://github.com/Yowkees/keyball/blob/main/keyball44/doc/rev1/buildguide_jp.md) バージョンを作成

## 使い方

以下の操作により Windows 起動時にスクリプトが有効化される

1. AutoHotkey v2.0 をインストール (<https://www.autohotkey.com/>)
2. 本リポジトリのクローン
3. `main.ahk` のショートカットを作成
4. `Win + R` で `shell:startup` を開き、作成したショートカットを貼り付け

## このリポジトリで実現できるカスタマイズ

### 数字キーの半角固定と効率化

* エンジニアとして活動する中で、全角数字を入力する機会はほぼ 0
* 都度、半角/全角キーを押すのは非効率 (例: 「～が 3 つ」と入力するのに半角/全角キーを 2 回も押下する必要がある)
* 日本語入力時にも半角数字が入力されるようカスタマイズ
* 全角入力時の「。」の入力について、以下のようにカスタマイズ
  * 直前の入力が数字の場合は「。」を「.」と半角にする ➡ 3.14 が切替なしで打てる
  * 直前の入力がそれ以外の場合は、そのまま「。」を入力する
  * 英語入力中は「.」を入力する

### その他

* エンジニアとして活動する中で、全角入力が必要ない以下のキーを半角固定

    ```plaintext
    ` _ # * ( ) |
    ```

### アプリ固有のカスタマイズ

* New Outlook (`olk.exe`) で `Ctrl+Enter` を無効化
  * 誤送信を防止し、意図的に送信ボタンを押すようにする

## 参考

必要のないカスタマイズの行は `main.ahk` の行頭に `;` を入力することでコメントアウトできるため、必要に応じて本リポジトリの特定のカスタマイズを無効化可能です。
